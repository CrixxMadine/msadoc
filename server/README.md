# msadoc server


## Description

Backend service to collect and manage `msadoc` metadata of microservices.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

The application is available on `localhost:3000`. Swagger UI serves the API documentation on route `/api`.


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
